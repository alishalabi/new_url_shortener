Ali's URL Shortener

The purpose of this project is the create a shortlink URL generator (similar to bitly). This is Ali's second iteration of a URL project.

Getting Started

Prerequisites

This project is built in Golang

Installing

`go get github.com/kataras/iris/v12@latest`

`go get -u github.com/etcd-io/bbolt`

`go get -u github.com/satori/go.uuid`

Built With

  • Iris: web framework
  • BBolt: embedded key/value database
  • : generates random shortlinks
